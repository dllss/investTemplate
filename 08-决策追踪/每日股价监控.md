## 🎯 每日监控检查清单

### 收盘后必做（每天16:30-17:00）

```markdown
□ 1. 获取4只标的收盘价
  ├── 京投交通科技（01522）：[ ]港元
  ├── 汇贤产业信托（87001）：[ ]港元
  ├── 天津发展（00882）：[ ]港元
  └── 华润医药（03320）：[ ]港元

□ 2. 计算当日涨跌幅
  ├── 对比昨日收盘价
  └── 计算涨跌百分比

□ 3. 更新组合净值
  ├── 股票市值 = 各标的股数 × 收盘价
  ├── 现金储备（不变，除非交易）
  └── 总净值 = 股票市值 + 现金

□ 4. 检查触发条件
  ├── 是否达到卖出条件？→ 记录并提醒
  ├── 是否下跌>5%（加仓机会）？→ 记录
  ├── 是否上涨>10%？→ 记录
  └── 是否异常波动（±15%）？→ 分析原因

□ 5. 记录到本文件
  ├── 添加新行到今日监控表格
  └── 更新历史记录

□ 6. 每周五额外操作
  ├── 计算周收益率
  ├── 更新周度监控概览
  └── 准备周报
```

### 自动计算工具（可选）

可使用Excel/Google Sheets建立自动计算：
```
股票市值 = SUM(持仓股数 * 实时股价)
总净值 = 股票市值 + 现金储备
日收益率 = (今日净值 - 昨日净值) / 昨日净值
累计收益率 = (当前净值 - 初始资金) / 初始资金
```
