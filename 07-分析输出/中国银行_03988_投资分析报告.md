银行业由于业务特性,**经营现金流指标不适用**:
- 银行主营业务是吸收存款和发放贷款
- 现金流量表中"经营活动现金流"主要受存贷款变动影响
- **不能用"经营现金流/净利润"评估银行质量**

**银行业核心指标**:
1. 净利润(直接反映盈利能力)
2. ROE/ROA(资本回报率)
3. 净息差(NIM,盈利能力核心)
4. 不良率+拨备覆盖率(风险指标)

### 4.2 净息差趋势分析(银行业核心)

#### 历史趋势

```
中国银行净息差(NIM)演变:
2020年: 1.60%

2022年: 1.47% (-8bp)
2023年: 1.45% (-2bp)
2024年: 1.40% (-5bp)
2025年: 1.26% (-14bp) 🔴 加速下降

5年累计下降: 34bp

```

| 因素 | 影响 | 说明 |

| 存款利率下调 | ✅ 正面 | 但降幅慢于贷款利率 |
| 竞争加剧 | 🔴 负面 | 银行间息差争夺 |
| 高收益资产占比下降 | 🔴 负面 | 地产贷款减少 |
| 低息存款占比提升 | ✅ 正面 | 活期存款增加 |

**综合判断**:🔴 净息差下降是行业性趋势,短期难以逆转。

### 4.3 周期拐点判断

**结论**:⚠️ 银行业尚未见底,中国银行处于**防御型配置阶段**,适合超长期持有,不适合追求短期估值修复。

## 第八章:高级烟蒂股分析框架

### 8.1 中国银行是否为烟蒂股?

**烟蒂股定义**:
- PB < 0.3倍
- 净现金 > 市值×2倍
- 清算价值 > 市值

**中国银行判定**:
- PB 0.53倍 > 0.3倍 ❌ **不符合**
- 银行业无"净现金"概念 N/A
- 清算价值难以估算(金融资产) N/A

**结论**:❌ **中国银行不是烟蒂股**,而是**低估值的纯硬收息型标的**。

### 8.2 为什么不适用烟蒂股框架?

**原因**:
1. 银行业净资产主要是金融资产(贷款、债券等),不是实物资产
2. 清算价值取决于资产质量(不良率),存在不确定性
3. 银行业是持续经营模式,不适合清算思维
4. PB 0.53倍虽低,但反映了盈利能力弱(ROE 8.94%)

**适用框架**:
- ✅ 纯硬收息型(央企银行)
- ✅ PB估值法(核心)
- ✅ 股息率估值法(辅助)
- ❌ 烟蒂股清算价值法(不适用)

- FCF≈净利润

**中国银行判定**:
- 银行业是**重资本模式**(需要大量资本金)
- 固定资产占比<5%(主要是金融资产)
- 边际成本不趋近于零(每笔贷款有风险成本)
- FCF概念不适用(现金流量表特殊)

**结论**:❌ **中国银行不属于轻资产模式**,是典型的**重资本金融业**。

### 10.2 银行业的特殊性

**商业模式**:
```
银行商业模式:
├── 吸收存款(负债端)
├── 发放贷款(资产端)
├── 赚取息差(净利息收入)
├── 中间业务(手续费收入)
└── 资本充足率约束(监管要求)
```

**与轻资产模式对比**:

| 特征 | 轻资产模式 | 银行业 |
|------|-----------|--------|
| 资本需求 | 低 | **高**(资本充足率约束) |
| 固定资产 | 低 | 低(但金融资产高) |
| 边际成本 | 趋近于零 | **不趋近于零**(风险成本) |
| 扩张方式 | 快速复制 | **受资本约束** |
| 估值方法 | PE/FCF | **PB/ROE** |

**结论**:银行业有独特的商业模式和估值方法,不适用第十章的轻资产模式框架。
