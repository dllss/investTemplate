## 一、项目简介

这是一个**港股/A股价值投资分析系统**，包含：

- 📖 **标准化分析模板**：10章完整分析框架
- 📊 **价格追踪工具**：自动监控关注标的股价
- 🎯 **模拟持仓管理**：记录投资决策和组合表现
- 🌐 **网页展示系统**：VitePress文档网站

**核心理念**：防御型价值投资，买得便宜才是硬道理（一级市场思维）

## 四、常用场景

### 场景 1：分析一只新股票

#### 第一步：快速初筛（5分钟）

打开 `README.md`，执行快速筛查：

| 检查项 | 排除标准 | 你的标的 |
|--------|---------|----------|
| ① 价格位置 | 距52周高点<30% | □ |
| ② 负债扫描 | 财务费用>5亿 | □ |
| ③ 现金覆盖 | 现金<市值20% | □ |
| ④ 盈利连续性 | 连续2年经营现金流为负 | □ |

**不通过** → 放入观察列表

#### 第二步：深度分析（90-150分钟）

1. 打开 `个股分析标准模版.md` 了解框架
2. 按章节深入分析：
   - 第一章：数据核查（用最新年报）
   - 第二章：央国企筛选与流派识别
   - 第三章：深度负债分析
   - 第四章：现金流趋势
   - 第五章：极端情景测试
   - 第六章：估值与安全边际
   - 第七章：决策与持仓管理

#### 第三步：输出报告

在 `07-分析输出/` 创建新文件：

**文件命名**：`{公司中文名}_{股票代码}_投资分析报告.md`

**示例**：
```markdown
# 工商银行 (01398.HK) 投资分析报告

**一句话结论**：🐢🍊 金龟标准，PB 0.4倍，股息率8%，FCF倍数2.5倍，建议买入

## 第一章：数据核查

### 1.1 基本信息
- 公司全称：中国工商银行股份有限公司
- 股票代码：01398.HK
- 报表期：2023年年报
- 数据来源：2023年年报原文（S级数据源）

...（按模板继续填写）
```

#### 第四步：记录决策

如果值得买入，记录到 `08-决策追踪/AI决策记录.md`

## 六、常见问题

### Q1: 价格追踪脚本无法获取数据？

**原因**：Yahoo Finance API不稳定，或股票代码格式错误

**解决**：
```bash
# 检查网络连接
ping finance.yahoo.com

# 确认股票代码格式：港股需要 .HK 后缀
# 正确：1398.HK
# 错误：01398 或 1398
```

### Q2: 网站启动失败？

**检查Node.js版本**：
```bash
node --version  # 需要 >= 18.0.0

# 版本过低？更新Node.js
# macOS: brew install node
# Windows: 从 nodejs.org 下载安装
```

**清理缓存重试**：
```bash
rm -rf node_modules package-lock.json
npm install
npm run docs:dev
```

### Q3: base_price 如何设置？

`auto_stock_tracker.py` 中的 `base_price` 首次运行时**自动设定为当天价格**。

**如果你想手动设定**（如设为历史低点）：

方法一：直接在代码中修改
```python
STOCKS = {
    '1398.HK': {
        'name': '工商银行',
        'base_price': 3.50,  # 手动设定基准价格
        ...
    }
}
```

方法二：编辑 `stock_data/tracking_data.json`
```json
{
  "1398.HK": {
    "base_price": 3.50,
    ...
  }
}
```

### Q6: 数据从哪里获取？

**数据源等级（强制执行）**：

| 等级 | 数据源 | 使用限制 |
|------|--------|----------|
| **S级** | 公司年报原文 | ✅ **必须使用**，禁止估算 |
| **A级** | 业绩公告 | ✅ 需与S级交叉验证 |
| **B级** | 东方财富等 | ❌ **不得用于核心财务数据** |
| **C级** | 财经媒体 | ⚠️ 仅供参考 |
| **D级** | AI估算 | ❌ **禁止用于投资决策** |

**获取年报方式**：
- **港股**：公司官网"投资者关系" → 财务报告
- **A股**：巨潮资讯网 (cninfo.com.cn)
- **便捷方式**：港交所披露易 (hkexnews.hk)

### Q7: macOS/Linux 如何运行 .bat 文件？

`.bat` 是Windows批处理文件，**不能在macOS/Linux运行**。

**解决方法**：
```bash
# 不运行.bat，直接运行Python脚本
cd 07-标的追踪
python3 auto_stock_tracker.py
```

如需定时任务（macOS）：
```bash
# 编辑crontab
crontab -e

# 添加定时任务（每周一至周五下午5点运行）
0 17 * * 1-5 cd /path/to/investTemplate/07-标的追踪 && /usr/bin/python3 auto_stock_tracker.py
```
