### 9.1 六维量化评分 [COMPUTABLE]

> **核心作用**：量化评估标的投资价值，避免主观偏差
> 
> **评分标准**：0-10分制，60分满分（银行业/金融业50分满分）

#### 标准评分表

| 维度 | 评分（满分10） | 说明 |
|------|----------------|------|
| **央企背景** | [0-10] | 10分:央企(国资委) / 7分:地方国企 / 5分:混合所有制 / 0分:民营 |
| **无有息负债** | [0-10] | 10分:财务费用<0 / 8分:有息负债<5亿 / 5分:有息负债<10亿 / 0分:有息负债>20亿 |
| **毛利率稳定性** | [0-10] | 10分:标准差<1% / 8分:<2% / 6分:<3% / 0分:>5% |
| **现金流质量** | [0-10] | 10分:OCF/净利润>150% / 8分:>120% / 6分:>100% / 0分:<80% |
| **估值吸引力** | [0-10] | 10分:FCF倍数<3倍 / 8分:<5倍 / 6分:<7倍 / 0分:>10倍 |
| **股息率** | [0-10] | 10分:>8% / 8分:>6% / 6分:>4% / 0分:<3% |
| **综合评分** | **[0-60]** | 评级说明见下表 |

**综合评级标准**：
- **55-60分**：🐢🍊 极品金龟级（立即建仓）
- **50-54分**：🐢🍊 金龟级（优先配置）
- **45-49分**：🐢💎 银龟级（可适度配置）
- **40-44分**：🐢🥉 铜龟级（观察为主）
- **<40分**：⚠️ 不符合标准（回避）

**特殊行业调整**：
- **银行业/金融业**：不考核"无有息负债"维度，满分50分
- **周期拐点型**：可接受适度负债（有息负债维度降权）
- **烟蒂股**：重点考核"估值吸引力"+"无有息负债"（双权重）

#### 示例：保利物业六维评分

| 维度 | 评分（满分10） | 说明 |
|------|----------------|------|
| 央企背景 | 10 | 国资委直管央企 |
| 无有息负债 | 10 | 财务费用为负 |
| 毛利率稳定性 | 9 | 标准差<1% |
| 现金流质量 | 8 | 经营现金流/净利润>150% |
| 估值吸引力 | 10 | FCF倍数2.33倍极品 |
| 股息率 | 6 | 4.5%略低于6%标准 |
| **综合评分** | **53/60** | **🐢🍊 金龟级** |

### 9.3 关键数字一句话 [MANUAL-ONLY]

> **格式要求**：用一句话总结核心财务数据+估值+投资建议，帮助快速决策

**模板**：
> **[公司名]当前市值[X]亿，[核心资产数据]，[核心估值指标]，[核心财务指标]，[主要优势]，[主要风险]。[投资建议一句话]。**

**示例1：保利物业**
> **保利物业当前市值177亿港元，账上现金119亿人民币（约130亿港元），剔除现金后真实业务估值仅47亿港元，年FCF约19亿人民币，FCF倍数仅2.33倍——不到2.5年回本，极品金龟级标的。**

**示例2：金融街物业（烟蒂股）**
> **金融街物业当前市值仅7.8亿人民币，净现金高达16.39亿（是市值的2.1倍），完全扣除现金后企业价值为负数，年FCF约1.2亿。股息率7.6%提供当下收益，2.1倍现金覆盖提供极强安全边际。但公司2024年业绩下滑9.4%，员工流失率20%，需警惕经营恶化。极品烟蒂股，5-8%仓位配置。**

**示例3：中国银行（银行业）**
> **中国银行当前市值1.50万亿人民币，PB 0.53倍深度破净，不良率1.23%为六大行最低，但股息率5.3%不符合熊市6%标准，派息下降6.6%，ROE 8.94%低于10%标准，净息差1.26%创历史新低。央企背景提供安全边际，但盈利能力持续承压。建议谨慎观望，等待股价<4.20港元（股息率>6%）或2026年年报确认派息企稳后再建仓。**

| 流派 | 买入标准 | 修复目标 | 卖出纪律 |
|------|----------|----------|----------|

### 9.6 "苹果买卖"估值修复模型（V5.5.6新增）⭐⭐⭐⭐⭐

> **实战来源**：价值投资本质阐述——以"苹果买卖"类比，价格显著低于价值时买入，修复至合理区间即止盈
> 
> **核心原则**：赚估值修复的钱（7-8毛），不贪泡沫的钱（1.5元）

#### 模型核心比喻

```
苹果正常价格：1.5元/斤（合理估值区间）
    ↓
买入点：0.3-0.4元/斤（显著低估，安全边际充足）
    ↓
卖出点：0.7-0.8元/斤（修复至合理区间下沿）
    ↓
不归零风险：即使苹果烂掉，自己吃也值0.5元（清算价值保底）
```

#### 映射到股票投资

| 苹果买卖 | 投资术语 | 海底捞案例映射 |
|---------|---------|--------------|
| 正常价格 | 合理PE/FCF倍数 | 15-20倍PE |
| 买入点（3-4毛） | 极端低估（🐢🍊金龟） | 15元以下（14-15倍PE） |
| 卖出点（7-8毛） | 估值修复完成 | 13-14元密集建仓区止盈 |
| 不归零风险 | 清算价值/净现金 | 品牌护城河+健康现金流 |

#### 实战纪律

```markdown
□ 买入纪律：只在"3-4毛"区间建仓（显著低估）
  ├── 海底捞：15元以下（14-15倍PE）
  ├── 华润啤酒：等待-12%回调后
  └── 原则：价格显著低于价值时买入

□ 持有纪律：忍受波动，等待修复
  ├── 散户应聚焦1-2年内可验证业绩
  ├── 避免与机构比拼5-10年远期预测
  └── 关注翻台率、门店数据等可验证指标

□ 卖出纪律：到"7-8毛"即止盈（不贪"1.5元"）
  ├── 此前海底捞13-14元密集建仓，19-20倍PE时止盈
  ├── 原因：当时价格已反映1年内业绩
  └── 存在更高性价比标的（如保利物业：170亿市值+140亿净现金）

□ 绝对禁止：在"1.2元"追入（错过就等下一轮）
  ├── 当前16.22元仅作试探（已非最佳买点）
  └── 宁愿错过，不可做错
```

#### 与"长期持有好公司"误区的对比

| 误区（❌） | 正解（✅） |
|-----------|-----------|
| "必须长期持有好公司" | 价格显著低于价值时买入，修复即卖出 |
| 不管价格，只看质地 | 质地再好，价格过高也不买 |
| 持有5-10年等成长 | 聚焦1-2年可验证业绩，估值修复即退出 |
| 与机构比拼远期预测 | 承认信息劣势，不做5年以上预测 |
| 赚泡沫的钱（1.5元） | 赚修复的钱（7-8毛），不贪最后一个铜板 |

#### 买入逻辑声明模板（苹果买卖版）

```markdown
### 买入逻辑声明（苹果买卖框架）

标的：[名称] [代码]

【苹果买卖定位】
- 正常价格（合理估值）：[如：15-20倍PE]
- 当前价格（买入点）：[如：16.22元，接近14-15倍PE区间]
- 买入性质：[如：插眼试探，非重仓]
- 目标修复区间：[如：13-14元密集区，19-20倍PE]
- 极端买点：[如：15元以下，具吸引力]

【一句话逻辑】
我以"苹果买卖"逻辑，在价格显著低于价值时（3-4毛）试探买入，
等待估值修复至合理区间下沿（7-8毛）即止盈，
绝不恋战追求泡沫顶部（1.5元）。

【风险控制】
- 总仓位上限：[如：核心15%]
- 加仓纪律：每跌3%加仓1000股，最多6-7次
- 止损条件：基本面恶化/估值修复逻辑破坏

【退出承诺】
□ 估值修复至目标区间即减仓/退出
□ 不因企业质地否定交易纪律
□ 当前仅保留极小仓位（约1万元），等待深度回调加仓
```
